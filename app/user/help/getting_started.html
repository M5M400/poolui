If you don't have a wallet create one

You can choose between online wallets and hardware wallets - each have their pros and cons.

Monero GUI Wallet: The safest way to store and secure your XMR, but it is your responsibility to keep your computer and thus XMR safe.

MyMonero: MyMonero is an online wallet maintained by a member of the core monero team, so while safe is still suceptable to hacks.

Offline wallet generator: Let's you generate a wallet with no internet connetion for better security. To later check the balance Monero GUI is best choice.

Pick a mining server and port

This pool uses PPLNS to determine payouts. It helps to combat pool hopping and ensures a good payout for miners.

For pool's url use pool.supportxmr.com

The Port is used in conjunciton with the url and is used to specify the starting difficulty. You should select this depending on the total hashing power of your mining rig. Also see config examples below.

Port descriptions:

- 3333 Low-end CPU
- 5555 Fast/Multi CPU
- 7777 GPU rigs
- 9000 SSL/TLS

If you can't get through firewall, try these:

- 80 Firewall bypass

The moment

By now you should have a wallet, mining software, pool url and port. It's time to put them together.

Here are some examples of the parameters that you need to pass on to your miner software, the format matters so be careful.

Depending on your miner you need to add the required parameters via command line or config.txt

Required fields are payment address and MinerIdentifier

MinerIdentifier is where you can set a name for your miner. Set email if you want to be able to log into the webpage to set options.

Basic:

Username : address
Password : MinerIdentifier

Advanced:

Username : address.paymentID+FixedDifficulty
Password : MinerIdentifier:Email
